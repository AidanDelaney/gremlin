Gremlin is a graph-based programming language. The documentation herein will provide all the information necessary to understand how to use Gremlin for graph query, analysis, and manipulation. Gremlin works over those graph databases/frameworks that implement the "Blueprints":http://blueprints.tinkerpop.com property graph data model.[1]

Please join the Gremlin users group at "http://groups.google.com/group/gremlin-users":http://groups.google.com/group/gremlin-users. Finally, if you are a Gremlin user, please add to the [[Gremlin in the Wild]] wiki page with your specific Gremlin uses cases.
