Gremlin is a "Turing-complete":http://en.wikipedia.org/wiki/Turing_completeness, "graph-based":http://en.wikipedia.org/wiki/Graph_%28mathematics%29 programming language developed in Java 1.6+ for key/value-pair multi-relational graphs known as property graphs. Gremlin makes extensive use of the "XPath 1.0":http://www.w3.org/TR/xpath language to support complex graph traversals. This language has applications in the areas of graph query, analysis, and manipulation. Connectors exist for the following data management systems:

"TinkerGraph":http://wiki.github.com/tinkerpop/gremlin/tinkergraph
"Neo4j":http://neo4j.org/ graph database
"Sesame 2.0":http://www.openrdf.org compliant RDF stores
"MongoDB":http://www.mongodb.org document database

The documentation for Gremlin can be found at this "location":http://gremlin.tinkerpop.com. Finally, please visit "TinkerPop":http://tinkerpop.com for other software products.
